# Cake Website 🎂

This is a simple frontend web development project created using HTML and CSS & Java script.

## 📌 Project Description
This website is designed to showcase different types of cakes with a clean and attractive layout. It includes images, basic navigation, and simple styling.

## ✨ Features
- Home page with cake showcase
- Cake image gallery
- Simple navigation menu
- Responsive design (basic)

## 🛠️ Technologies Used
- HTML
- CSS
- Java script

## 📁 Project Structure
- index.html
- images/ (folder for cake images)

## 🚀 How to Run
1. Download or clone the repository
2. Open 'index.html' in any browser
